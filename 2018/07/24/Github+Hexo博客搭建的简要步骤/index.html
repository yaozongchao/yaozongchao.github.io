    • Github+Hexo博客搭建的简要步骤

    部署前准备

    1. 准备github账号，新建仓库，命名为yaozongchao.github.io
    2. 本地安装node.js，用node -v命令检查是否本地已安装
    3. 本地安装hexo，sudo npm install hexo-cli -g
    4. 进入本地合适的文件夹，创建博客目录hexo init yaozongchao.github.io
    5. cd yaozongchao.github.io
    6. npm install
    7. 此时使用hexo g

    将本地的博客部署到github上

    1.在yaozongchao.github.io目录下执行sudo npm install hexo-deployer-git --save，安装部署插件

    1. 修改本地根目录下的_config.yml文件，将其中deploy的部分修改为

    2. 执行hexo d，将本地博客部署到github上，此时用浏览器登录https://yaozongchao.github.io即可浏览博客

    美化博客

    1. 在https://hexo.io/themes/中寻找钟意的主题
    2. 在本地目录的themes目录下执行git clone git@github.com:Ben02/hexo-theme-Anatole.git，将主题文件下载下来
    3. 修改_config.yml文件，将其中theme: hexo-theme-Anatole改为相应的主题名。当然，也有部分主题需要严格按照github作者的描述来安装
    4. hexo clean
    5. hexo g -d生成完直接部署

    操作博客

    1. 在本地source/_posts文件夹下新建md文件，就是新建文章了，使用markdown语法，title一定要有，否则生成不会成功

    2. hexo g -d
